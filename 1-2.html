1. HTML 제목은 <h1> ~ <h6> 태그로 정의 됩니다.

<h1>은 가장 중요한 제목을 <h6>은 가장 중요하지 않은 제목을 정의합니다:

This is heading 1

This is heading 2

This is heading 3

2. 단락은 <p> 태그로 정의됩니다.

This is paragraph

This is another paragraph

3. 링크는 <a> 태그로 정의됩니다.

this is safehomes link

4. 이미지는 <img> 태그로 정의됩니다. 소스파일(src), 대체 텍스트(alt), 너비 및 높이가 속성으로 제공됩니다:

장바구니목록

    사과 사과 사과
    사과 사과 사과
과일 수량
